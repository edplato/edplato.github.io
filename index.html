Flick Pick - Movie Search

RESTful Web Application
NodeJs / MongoDB / Express / JavaScript / Bootstrap

This MVC application was created to help narrow down movie choices by allowing the user to search movies, retrieve detailed information, provide IMDB links, and save movies to watch for later. It uses routed API's to search movie data, parse JSON results to a temporary movie pool(anonymously or logged in), and store client associated data. This RESTful application was built on MongoDB mLab for data storage, Express(with EJS), and Node.js.
